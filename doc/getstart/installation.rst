Installation
============

Requirements
------------

* python >= 3.6
* pytorch >= 1.6 (install `here <https://pytorch.org/>`_)

Installation
------------

In your terminal, type:

.. code-block::

    pip install xitorch

Or if you want to install from source, type:

.. code-block::

    git clone https://github.com/mfkasim1/xitorch/
    cd xitorch
    pip install -e .
